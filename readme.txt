== Description ==

Pro Blog Stats provides a single location to monitor the performance and statistics for your blog. It gathers statistics from many sources and shows your Overall Performance Chart as a measure of how many statistics have changed in a positive direction compared to the previous period.

Monitoring the right statistics can make or break your blog's success.

How are you measuring the success of your blog?

Traffic?
Length of stay i.e. Bounce Rate?
RSS Subscribers?
Number of Comments?
Backlinks?
Social Bookmarking Numbers?
Pages Indexed?
Others?
Or a myriad of the above?

The problem is accessing the data quickly, easily, and in one place. You have to constantly log in and out of numerous statistical services... and that's time consuming and a right pain.

With Pro Blog Stats you get to see all the stats you need in one place your WordPress dashboard with one click.

You will need an [Pro Blog Stats API Key](http://problogstats.com/signup/). Signup here (http://problogstats.com/signup/) to get your API key.

== Installation ==

Upload the Pro Blog Stats plugin to your blog, Activate it. You will be asked to enter your [Problogstats.com API key](http://problogstats.com/signup/).
In the admin sidebar, there will be a new menu called 'Pro Blog Stats'. You can select the setup option to setup
your [Problogstats.com API key](http://problogstats.com/signup/) and service information.

Thats it.. You are done!
